# 🎬 Movie Recommendation Analytics System

## About
A SQL-based movie analytics system built with MySQL that simulates 
a real-world streaming platform. The database contains movies from 
diverse film industries including Tamil, Chinese, English, German, 
Thai, Korean, Japanese, and other international cinemas.

## SQL Concepts
- Joins → INNER JOIN, LEFT JOIN, Multiple JOINs
- Aggregate Functions → COUNT, AVG, GROUP BY, HAVING
- Window Functions → DENSE_RANK, PARTITION BY
- Common Table Expressions (CTEs)
- Stored Procedures with Parameters
- Triggers → Auto track user activity
- Indexing & Query Optimization
- Subqueries

## Database
- 6 Tables
- 3014+ Records
- 12+ International Film Industries
- 226 Movies | 512 Users | 1020+ Watch Records

## Features
- 🎯 Movie genre analytics
- 👤 User watch history tracking
- ⭐ Movie rating & ranking analysis
- 📊 Genre popularity report
- 🏆 Most active user tracking
- ⚡ Query performance optimization
